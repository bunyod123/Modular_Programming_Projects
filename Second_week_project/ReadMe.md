# Biz bu scrapped qilingan dataset dunyo boyicha 1 oylik zilzilaga oid malimotlar yegilgan
## Bu data setdagi malumotlarga asoslanib zilzilani xavsiz, xavli va juda xavli qismini bashorat qilamiz

# Bu datasetda ishlatilgan so'zlar:

Lat. degrees – Kenglik (graduslarda)
(Geografik kenglik koordinatasi)

Lon. degrees – Uzunlik (graduslarda)
(Geografik uzunlik koordinatasi)

Depth km – Chuqurlik (km)
(Zilzila o‘chog‘ining yer ostidagi chuqurligi)

Region – Hudud / Mintaqa
(Zilzila sodir bo‘lgan geografik joy)

Type – Turi
(Zilzila turi, masalan: tektonik, vulqonli va h.k.)

A/M – Avtomatik / Qo‘lda (Manual)
(Ma’lumot avtomatik tizim tomonidan aniqlanganmi yoki mutaxassis tomonidan tekshirilganmi)

Magnitude – Magnituda (kuchi)
(Zilzilaning energiya miqdori, masalan Richter shkalasi bo‘yicha)

Network – Tarmoq (Seysmik kuzatuv tarmog‘i)
(Ma’lumotni qayd qilgan seysmologik stansiyalar tarmog‘i)
